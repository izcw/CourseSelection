<h1 align="center">学生选课管理系统</h1>

## 简介

基于vue3、Element-Plus、JavaWeb、MySQL技术实现的学生选课管理系统

## 使用

### 前端

`/Frontend`目录

- 获取代码

```sh
git clone https://github.com/izcw/CourseSelection.git
```

- 安装依赖

```sh
yarn install
```

- 运行

```sh
yarn serve
```

- 打包

```sh
yarn build
```

- 本地预览

```sh
yarn preview
```

- 打包/预览

```sh
yarn build:preview
```

### 后端

`/Backend`目录
